                                        .main_banner (O) /* 언더바로 단어와 단어사이를 잇는다. */
                                        .mainBanner (X)

                                        .btn_large (O) /* px크기 말고 소/중/대 로 설정 */
                                        .btn_300 (X)

                                        .font_warning (O) /* red의 의미 */
                                        .font_red (X)

                                        .email_group (O) /* 소문자로 시작 */
                                        .Email_group (X)

                                        .btn_primary_02_on (O) /* 형태_의미_순서_상태 */
                                        .btn_02_primary_on (X)
  • - 스타일 적용을 위한 클래스는 단어와 단어 사이를 언더바(_)로 구분한다.
  • - 아이디/클래스 이름은 용도를 나타내는 단어를 사용하되, 크기/색상 등의 스타일을 특정짓는 단어를 지양한다.
  • - 이름은 영문 대문자, 숫자, 특수문자로 시작할 수 없다.
  • - 네이밍의 조합은 '형태_의미_순서_상태'을 기본 순서로 사용한다.

css 코드 작성규칙

기본규칙
                                        @charset "utf-8";
                                        /* ============================================================
                                            파일정의 : style.css
                                            속성순서 : 레이아웃 > 크기/간격 > 박스모양 > 정렬/폰트 > 기타 
                                        ============================================================ */

                                        /* 세로쓰기 */
                                        .box_common {
                                            display: flex;
                                            width: 100%;
                                            height: auto;
                                            padding: 20px;
                                            margin: 20px;
                                            border-radius: 20px;
                                            background-color: #f5f5f5;
                                            text-align: center;
                                            font-size: 1em;
                                            color: #999;
                                            overflow: hidden;
                                        }

                                        /* 가로쓰기 */
                                        .box_common {display: flex; width: 100%; height: auto; padding: 20px; margin: 20px; border-radius: 20px; background-color: #f5f5f5; text-align: center; font-size: 1em; color: #999; overflow: hidden;}
  • - 마지막 선언된 속성에도 세미콜론(;)을 사용한다.
  • - 세로쓰기/가로쓰기는 통일되어야 한다. 예를들어 하고있는 프로젝트에서 세로쓰기로 css를 작성중이면 모든 css는 세로쓰기로 통일 되어야 한다.
  • - 종료 주석은 작성하지 않는다.
  • - 공통선택자는 웹페이지의 성능저하를 가져오기 때문에 가급적 사용하지않는다. (*)
사용자 지정속성(변수)
                                        @charset "utf-8";
                                        /* ============================================================
                                            파일정의 : reset.css
                                            지정 속성 적용 예제
                                        ============================================================ */
                                        /* reset.css */
                                        
                                        :root {
                                            /* color */
                                            --main-color: brown;
                                            --sub-color: brown;
                                            --main-bg-color: brown;
                                            --sub-bg-color: brown;

                                            /* font size */
                                            --font-xs: 1.2rem;
                                            --font-s: 1.4rem;
                                            --font-m: 1.6rem;
                                            --font-l: 2rem;
                                            --font-xl: 2.4rem;
                                          }
                                        
                                        /* style.css */
                                        element {
                                            /* color */    
                                            background-color: var(--main-color);
                                            background-color: var(--main-bg-color);
                                            /* font size */
                                            font-size: var(--font-xs);
                                            font-size: var(--font-s);
                                        }
  • - 사용자 지정 속성은 전용 표기법을 사용해 정의하고 var() 함수를 사용해 접근할 수 있다.
  • - 사용자 지정 속성은 두 개의 붙임표로 시작하는 속성의 이름과 함께 유효한 값이라면 그 값으로 지정해 사용한다.
  • - 사용자 지정 속성은 종속 대상이며 부모로부터 상속된다.
  • - 참고: 사용자 지정 속성의 이름은 대소문자를 구분한다. --my-color와 --My-color는 서로 다른 속성으로써 처리한다.
  • - 주의: ie에서 지원되지 않으므로 첨부된 root.js 파일을 다운받아 대응한다.

미디어쿼리
                                        /* ============================================================
                                            media query
                                        ============================================================ */
                                        /* width 768px 이하 */
                                        @media screen and (max-width: 768px) {
                                            /* 작성한 코드그룹 이름 */
                                        }

                                        /* width 1200px 이하 */
                                        @media screen and (max-width: 1200px) {
                                            /* 작성한 코드그룹 이름 */
                                        }
  • - 미디어쿼리 2개이상 영역을 작성할땐 기본 css 주석이름을 그대로 적용한다.(ex)header 반응형 작업시 주석에 header로 통일)
  • - 미디어쿼리 모바일 기준점은 768px로 최하 360px로 작업한다

폰트 스타일

글자의 크기와 폰트
  • - 16px 크기를 기본 폰트로 사용한다.
  • - 모바일 반응형의 경우 14px/0.9em 을 사용한다.
글자 기본 색상
  • - 기본적으로 디자인에 지정되어있는 색상을 쓰며, 정해져있지 않다면 기본 #222 color를 사용한다.
